Eye See Mobile

Working Features
=================
1. Double tap screen to get verbal response back from the app
2. The application is hardcoded to say "This is a prototype" when clicked.
The final version will verbally describe the response from our trained model.

Non Working Features
===================
1. Trained model is not created
2. Responses for what the camera is seeing outside of "This is a prototype" 

Using The Prototype
===================

1. Launch App by deploying to device (physical)
2. Provide the application with camera privileges 
3. Point camera at object
4. Double tap screen to get response from app
5. Listen for what the automated verbal response
